# 🏥 Balaji Clinic Website

## 📌 Project Description

This project is a **responsive clinic website** built using **HTML, Tailwind CSS, and Vanilla JavaScript**.
It is designed to provide basic information about a clinic, its services, and allow users to book appointments.

---

## 🚀 Features

* ✅ Responsive design (mobile, tablet, desktop)
* ✅ Modern UI using Tailwind CSS
* ✅ Hamburger menu (mobile navigation)
* ✅ Interactive menu toggle (hamburger → cross)
* ✅ Appointment booking form
* ✅ Smooth hover and transition effects

---

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS (CDN)
* JavaScript (Vanilla JS)

## 📱 Responsive Design

The website is fully responsive using Tailwind’s breakpoints:
For Example:
* Mobile → Hamburger menu
* Desktop → Full navigation bar
